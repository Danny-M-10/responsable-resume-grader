LOGO SETUP INSTRUCTIONS

The application now uses a fixed company logo for consistency across the UI and PDF reports.

To set up the logo:
1. Ensure the file `responsableLOGO-color-2048px.jpg` is present in the project root:
   /Users/danny/Documents/Cursor/Projects/crossroads_Candidate_Ranking_Application/responsableLOGO-color-2048px.jpg

2. The logo will automatically appear in the UI header and all PDF report headers (left-justified)

3. Logo specifications:
